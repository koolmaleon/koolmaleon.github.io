Work with Terminal on your Mac

Yesterday I restored my Mac to factory, because I felt I must have messed up something for I had been trying out a lot of fancy stuffs with sudo while I was not sure what those were. Besides, my girlfriend got a new Mac recently, so I planed to go through the set-up process and then teach her with confidence. Isn’t that sweet of me? XD

Apps Recommendation Before Setting-up Terminal

Well, before setting up terminal, I have some subjective recommendations for Apps. I think these Apps helps a lot in using Mac in a programmer’s way.

Alfred

Alfred is a substitution, or I should say, an upgradation for Spotlight. With Alfred, you can easily open apps, find files, do web searches and more with commands. Not much to say, just try it.

iTerm2

This is a substitution for the original Terminal shipped with ths Mac OS. I didn’t do much investigation on this, because I simply use it as a substitution for tmux. In a nutshell, you can create tabs, split screens and switch between these windows with friendly shortcuts.

Set up your terminal

Now try to use Alfred to start iTerm, and let’s start setting up our terminal!

Recommended Plugins

Plugins for Vim are like equipments for warriors, without which, their power will be greatly weakened and their potential unexploited. Here, I just recommend some plugins I use, and I will only briefly introduce them. For new-comers to Vim, the basic operations and commands should be mastered before trying out these powerful extensions.

  • YouCompleteMe: “YouCompleteMe is a fast, as-you-type, fuzzy-search code completion engine for Vim.”, as said in the project’s homepage.
  • Syntastic: Runtime syntax checker.
  • NerdTree: Powerful sidebar for working space and file management.
  • Tagbar: Powerful sidebar for tags, functions, and classes navigation.
